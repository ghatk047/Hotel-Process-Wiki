Description

VIP and Celebrity Security Protocol ensures the safety and security of high-profile guests during their stay at a Marriott full-service hotel.

Risks

  • ▸ Overlooking VIP/Celebrity Security Alert in Oracle OPERA Cloud
  • ▸ Neglecting security team assignment and coordination with local law enforcement
  • ▸ Inadequate room security measures
  • ▸ Poor monitoring of public areas
  • ▸ Missing background checks for VIP/Celebrity guests
