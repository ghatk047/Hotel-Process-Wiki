ℹ️ Process Attributes

Identification

  • ID SC-SE-07
  • Domain Safety, Security & Compliance
  • Group Security Operations
  • Process Security Incident Reporting
  • Trigger Detection of a security breach or suspicious activity by any staff member, guest, or external party.
  • Outcome The incident is logged, investigated, reported to the appropriate authorities, and documented for future reference. The hotel's security protocols are reviewed and improved if necessary.

KPIs

  • ▸ Incident identification within 2 minutes
  • ▸ Management notification within 5 minutes
  • ▸ Internal investigation initiation within 10 minutes
  • ▸ Detailed incident documentation within 30 minutes
  • ▸ Police report submission within 1 hour
  • ▸ Guest notification within 2 hours

Description

This process outlines how to report a security incident at a Marriott full-service hotel using Oracle OPERA Cloud PMS and other key systems.

Risks

  • ▸ False positives leading to unnecessary investigations
  • ▸ Delays in communication affecting incident response time
  • ▸ Limited access to surveillance footage hindering the investigation
  • ▸ Complex incidents requiring thorough documentation
  • ▸ Coordination with local law enforcement being challenging
