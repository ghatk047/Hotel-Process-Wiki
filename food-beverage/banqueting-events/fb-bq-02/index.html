ℹ️ Process Attributes

Identification

  • ID FB-BQ-02
  • Domain Food & Beverage Operations
  • Group Banqueting & Events Catering
  • Process Banquet Room Setup & Layout
  • Trigger Receiving a confirmed reservation for a banquet event
  • Outcome Banquet room is fully set up and ready for use with all necessary equipment, decorations, and services provided according to the client's specifications.

KPIs

  • ▸ Room assignment confirmation within 3 minutes
  • ▸ Setup completion checklist finalized within 25 minutes
  • ▸ Client notification sent within 3 minutes
  • ▸ Less than 10% of reservations have incorrect or missing information
  • ▸ Less than 5% of amenity requests result in mismatches

Description

This process outlines the steps involved in setting up and laying out banquet rooms at a full-service Marriott hotel using Oracle OPERA Cloud PMS.

Risks

  • ▸ Incorrect reservation details leading to setup errors
  • ▸ Supply chain delays causing equipment shortages
  • ▸ Technical issues with banquet room equipment
  • ▸ Client dissatisfaction due to final setup
  • ▸ System connectivity problems affecting updates
