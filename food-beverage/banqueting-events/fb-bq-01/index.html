Identification

  • ID FB-BQ-01
  • Domain Food & Beverage Operations
  • Group Banqueting & Events Catering
  • Process Event Enquiry & BEO Creation
  • Trigger Receiving an event inquiry via phone, email, or online booking system
  • Outcome Successful creation of a BEO with all necessary details for the event

KPIs

  • ▸ Less than 3 minutes per event inquiry
  • ▸ Less than 10 minutes per initial client contact
  • ▸ Less than 15 minutes per BEO creation
  • ▸ Less than 2 hours per order placement

Description

The process involves receiving an event inquiry from a client and creating a Banquet Event Order (BEO) in Oracle OPERA Cloud PMS.

Risks

  • ▸ Client dissatisfaction with proposed changes
  • ▸ Supply chain disruptions affecting event supplies
  • ▸ Handling last-minute changes to event details
  • ▸ Managing client expectations for event capacity
  • ▸ Ensuring all required fields are accurately filled in BEO creation
