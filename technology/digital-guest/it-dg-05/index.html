Description

This process outlines the management of digital signage and wayfinding systems in a full-service Marriott hotel using Oracle OPERA Cloud PMS and other key Marriott systems.

Risks

  • ▸ Coordination issues with multiple stakeholders during project initiation
  • ▸ Integration problems between different Marriott systems
  • ▸ Delays in content approval and quality assurance
  • ▸ Technical challenges during system setup and configuration
  • ▸ Staff resistance to adopting new digital signage systems
