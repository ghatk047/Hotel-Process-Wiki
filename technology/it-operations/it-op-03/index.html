ℹ️ Process Attributes

Identification

  • ID IT-OP-03
  • Domain Technology & Systems
  • Group IT Operations & Infrastructure
  • Process POS System Management
  • Trigger Daily opening, end-of-day closing, or any significant change in POS system configuration
  • Outcome Successful completion results in operational readiness of the POS system for the next day's transactions with minimal downtime and errors.

KPIs

  • ▸ System uptime greater than 99%
  • ▸ Less than 5 minutes for system check and final readiness confirmation
  • ▸ Less than 1 hour per person for staff training
  • ▸ Less than 2 hours for log review and issue resolution planning

Description

This process outlines the management of Point of Sale (POS) systems in a full-service Marriott hotel using Oracle OPERA Cloud PMS. It ensures smooth transaction processing and maintains system integrity.

Risks

  • ▸ Hardware failures leading to extended downtime
  • ▸ Software update compatibility issues disrupting operations
  • ▸ Incorrect POS configuration causing operational errors
  • ▸ Staff resistance or lack of understanding during training
  • ▸ Overlooked error messages in logs resulting in system failures
