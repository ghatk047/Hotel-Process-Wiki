KPIs

  • ▸ Onboarding Plan creation within 2 days
  • ▸ Property Manager assignment within 1 day
  • ▸ PMS configuration completed in less than 5 days
  • ▸ Staff training completion within 2 weeks

Description

The process outlines the steps involved in onboarding a new franchise agreement and setting up a property within a Marriott full-service hotel using Oracle OPERA Cloud PMS.

Risks

  • ▸ Delays in obtaining necessary approvals from corporate headquarters
  • ▸ Technical issues during system configuration
  • ▸ Integration challenges between PMS and RMS systems
  • ▸ Data entry errors during initial setup of CRM
  • ▸ Unexpected system bugs during testing
